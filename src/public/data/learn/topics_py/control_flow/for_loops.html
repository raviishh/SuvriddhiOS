For Loops in Python

A for loop is used for definite iteration — when you want to iterate over a sequence of elements. It is cleaner, safer, and often more readable than a while loop.

1. Basic Syntax

A for loop iterates over an iterable (like a list, string, or range). 

numbers = [1, 2, 3, 4]

for num in numbers:
    print(num)

Execution flow:

  • Take first element
  • Execute block
  • Move to next element
  • Repeat until exhausted

Core Idea
A for loop automatically handles iteration control. No manual counter required.

2. Using range()

range() generates a sequence of numbers. 

for i in range(5):
    print(i)

Output: 0, 1, 2, 3, 4

Variations: 

range(stop)
range(start, stop)
range(start, stop, step)
for i in range(1, 10, 2):
    print(i)

Prints odd numbers from 1 to 9.

3. Looping Through Strings

Strings are iterable character by character. 

for letter in "Python":
    print(letter)

4. Using break and continue

break exits the loop immediately. continue skips to the next iteration. 

for i in range(5):
    if i == 3:
        break
    print(i)
for i in range(5):
    if i == 2:
        continue
    print(i)

5. Using else with For

A for loop can include an else block. It executes if the loop completes without hitting break. 

for i in range(3):
    print(i)
else:
    print("Loop completed normally")

6. Nested For Loops

You can place a for loop inside another. 

for i in range(3):
    for j in range(2):
        print(i, j)

Common use cases:

  • Matrix operations
  • Grid-based games
  • Combinatorial logic

Performance Note
Nested loops multiply time complexity. Two loops → roughly O(n²).

7. Looping with Index: enumerate()

Use enumerate() to access both index and value. 

fruits = ["apple", "banana", "cherry"]

for index, fruit in enumerate(fruits):
    print(index, fruit)

8. Common Mistakes

  • Modifying a list while iterating over it
  • Off-by-one errors with range()
  • Overusing nested loops
  • Using while when a for loop is simpler
Quick Check
What prints? for i in range(1, 5): print(i * 2) How many times does it run?

9. Why For Loops Matter

For loops are the standard tool for iteration in Python. They are concise, readable, and less error-prone than manual counter loops.

Mastering iteration is fundamental to processing data, building algorithms, and writing efficient programs.

Practice Strategy
Rewrite while loops as for loops when possible. Then analyze readability and complexity. This builds structural awareness in your code.
