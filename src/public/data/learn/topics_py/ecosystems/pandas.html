Pandas Basics

A simple introduction to Pandas — the library used to work with tables and data in Python. Think of Pandas as Excel inside Python.

1. What is Pandas?

Pandas is a Python library used to work with structured data. Structured data means data in rows and columns, like a table.

Analogy
If NumPy is a powerful calculator, Pandas is a smart spreadsheet. It helps you organize, filter, and analyze data easily.

Pandas is used in data science, business analysis, and machine learning. It is built on top of NumPy.

2. Installing and Importing Pandas

Install Pandas using pip: 

pip install pandas

Import it in Python: 

import pandas as pd

We use pd as a short name.

3. Creating a DataFrame

The main object in Pandas is called a DataFrame. A DataFrame is a table. 

import pandas as pd

data = {
    "Name": ["Ali", "Sara", "John"],
    "Marks": [85, 90, 78]
}

df = pd.DataFrame(data)
print(df)

Output:

Name Marks
Ali 85
Sara 90
John 78

Each column has a name. Each row is one record.

4. Reading a CSV File

CSV means "Comma Separated Values". It is a simple file format for storing tables. 

df = pd.read_csv("students.csv")
print(df)

Pandas reads the file and turns it into a DataFrame.

5. Selecting Data

You can select one column: 

print(df["Marks"])

You can filter rows using a condition: 

high_marks = df[df["Marks"] > 80]
print(high_marks)

This shows only students with marks greater than 80.

6. Basic Data Information

Pandas has useful functions to understand data. 

print(df.head())       # First 5 rows
print(df.describe())   # Summary statistics
print(df["Marks"].mean())

describe() gives average, minimum, maximum, and more.

7. Why Pandas Matters

Pandas is used in:

  • Data analysis
  • Business reports
  • Machine learning
  • Research projects

It makes working with large data simple and fast.
