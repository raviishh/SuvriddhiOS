Matplotlib Basics

A simple introduction to Matplotlib — the library used to turn numbers into pictures. Think of Matplotlib as a drawing tool for your data.

1. Why Do We Visualize Data?

Numbers are important. But pictures are easier to understand. A graph can show patterns quickly.

Analogy
Reading many numbers is like reading a long paragraph. A graph is like seeing a picture. Your brain understands it faster.

Matplotlib helps us create graphs in Python.

2. Installing and Importing Matplotlib

Install Matplotlib using pip: 

pip install matplotlib

Import it in Python: 

import matplotlib.pyplot as plt

We use plt as a short name.

3. Creating a Line Plot

A line plot shows how something changes over time. 

import matplotlib.pyplot as plt

x = [1, 2, 3, 4]
y = [10, 20, 15, 25]

plt.plot(x, y)
plt.show()

plt.plot() creates the line.
plt.show() displays the graph.

4. Adding Titles and Labels

A good graph always has a title and labels. 

plt.plot(x, y)

plt.title("Sales Over Time")
plt.xlabel("Month")
plt.ylabel("Sales")

plt.show()

Titles and labels make the graph clear.

5. Creating a Bar Chart

A bar chart compares values. 

students = ["Ali", "Sara", "John"]
marks = [85, 90, 78]

plt.bar(students, marks)
plt.title("Student Marks")
plt.xlabel("Student")
plt.ylabel("Marks")

plt.show()

plt.bar() creates bars instead of a line.

6. Saving a Graph

You can save the graph as an image file. 

plt.plot(x, y)
plt.title("Example Graph")

plt.savefig("graph.png")
plt.show()

This saves the graph as a PNG file.

7. Why Matplotlib Matters

Matplotlib is used in:

  • Data science
  • Research papers
  • Business reports
  • Machine learning projects

Clear graphs help people understand your results.
