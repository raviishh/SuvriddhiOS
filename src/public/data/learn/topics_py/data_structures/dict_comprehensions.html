Dictionary Comprehensions in Python

Dictionary comprehensions provide a concise way to construct dictionaries. They combine iteration, transformation, and optional filtering into a single expressive statement.

1. What Is a Dictionary Comprehension?

A dictionary comprehension creates a dictionary using a compact syntax. 

squares = {x: x * x for x in range(5)}
print(squares)

Equivalent traditional approach:

squares = {}
for x in range(5):
    squares[x] = x * x
Core Structure
{key_expression: value_expression for item in iterable}

2. Adding Conditions

You can filter items using an if clause.

even_squares = {x: x * x for x in range(10) if x % 2 == 0}
print(even_squares)

Only items meeting the condition are included.

Extended Structure
{key: value for item in iterable if condition}

3. Transforming Existing Dictionaries

Dictionary comprehensions are commonly used to transform dictionaries. 

prices = {"apple": 100, "banana": 50, "cherry": 200}

discounted = {item: price * 0.9 for item, price in prices.items()}
print(discounted)

You can also swap keys and values.

inverted = {value: key for key, value in prices.items()}
print(inverted)

4. Conditional Value Assignment

Inline conditional expressions can modify values.

grades = {"Alice": 85, "Bob": 72, "Charlie": 90}

status = {
    name: "Pass" if score >= 75 else "Fail"
    for name, score in grades.items()
}

print(status)

5. Nested Dictionary Comprehensions

Comprehensions can generate nested structures.

matrix = {
    i: {j: i * j for j in range(3)}
    for i in range(3)
}

print(matrix)

Use nested comprehensions cautiously to preserve readability.

6. Readability Guidelines

  • Keep expressions simple and clear
  • Avoid deeply nested logic
  • Prefer explicit loops when clarity decreases
Best Practice
Prioritize maintainability over compactness.

7. Common Mistakes

  • Forgetting the colon between key and value
  • Confusing dictionary and set comprehensions
  • Overcomplicating nested comprehensions
  • Creating duplicate keys unintentionally
Quick Check
What is the output? result = {x: x % 2 for x in range(4)}

8. Why Dictionary Comprehensions Matter

Dictionary comprehensions allow declarative transformation of structured data.

They are especially powerful when processing configuration data, JSON-like structures, and algorithmic mappings.

Practice Strategy
Transform a dataset by filtering entries and modifying values using dictionary comprehensions.
