Working with APIs in Python (requests)

In this lesson, you will learn how to connect your Python program to real-world data on the internet. By the end, you will build a simple Weather App that fetches live data from an API.

1. What is an external API?

An external API is a server built by someone else. You send it a request. It sends you data back.

Simple idea
Your program asks a question. Another server answers with data.

Many companies provide APIs:

  • Weather services
  • Maps
  • News providers
  • Payment systems

2. Installing the requests library

The requests library makes HTTP requests simple. 

pip install requests

Now Python can talk to the internet.

3. Making your first API request

import requests

response = requests.get("https://api.github.com")

print(response.status_code)
print(response.text)

Explanation:

  • requests.get() sends a GET request.
  • status_code tells if it worked.
  • text shows the raw response.

Status codes
200 → Success 404 → Not found 500 → Server error

4. Working with JSON data

Most APIs return JSON. Instead of reading raw text, we convert it into a Python dictionary. 

import requests

response = requests.get("https://api.github.com")

data = response.json()

print(data)

Now you can access values like this: 

print(data["current_user_url"])

5. API keys (important concept)

Some APIs require a special key. This is called an API key.

It proves who you are.

Important
Never share your API key publicly.

Example: 

params = {
    "q": "London",
    "appid": "YOUR_API_KEY"
}

response = requests.get("https://api.openweathermap.org/data/2.5/weather", params=params)

6. Building a Weather App

import requests
import sys

city = sys.argv[1]

API_KEY = "YOUR_API_KEY"

params = {
    "q": city,
    "appid": API_KEY,
    "units": "metric"
}

response = requests.get(
    "https://api.openweathermap.org/data/2.5/weather",
    params=params
)

if response.status_code == 200:
    data = response.json()
    temperature = data["main"]["temp"]
    condition = data["weather"][0]["description"]

    print("City:", city)
    print("Temperature:", temperature, "°C")
    print("Condition:", condition)
else:
    print("Error fetching weather data.")

Run: 

python weather.py London

7. Handling errors safely

Internet requests can fail. Always handle errors. 

try:
    response = requests.get("https://api.github.com")
    response.raise_for_status()
    data = response.json()
    print(data)
except requests.exceptions.RequestException as e:
    print("Something went wrong:", e)
