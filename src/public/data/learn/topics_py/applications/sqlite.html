Intro to Databases with SQLite

In this lesson, you will learn how to store data properly using a real database. By the end, you will build a small User Manager that saves information in SQLite.

1. Why do we need databases?

So far, we stored data in memory (lists). The problem:

  • Data disappears when the program stops
  • It becomes hard to manage large amounts of data
Simple idea
A database is a safe storage system for your data.

Real applications always use databases.

2. What is SQLite?

SQLite is a small database built into Python.

  • No installation needed
  • Data stored in a single file
  • Very simple to use

We use the built-in module: sqlite3

3. Creating a database

import sqlite3

# connect to database (creates file if it does not exist)
conn = sqlite3.connect("users.db")

# create a cursor (used to execute SQL)
cursor = conn.cursor()

print("Database created successfully!")

conn.close()

A new file called users.db will appear.

4. Creating a table

import sqlite3

conn = sqlite3.connect("users.db")
cursor = conn.cursor()

cursor.execute("""
CREATE TABLE IF NOT EXISTS users (
    id INTEGER PRIMARY KEY,
    name TEXT,
    age INTEGER
)
""")

conn.commit()
conn.close()

print("Table created!")

Explanation:

  • A table stores data in rows and columns
  • PRIMARY KEY gives each row a unique ID

5. Inserting data (INSERT)

import sqlite3

conn = sqlite3.connect("users.db")
cursor = conn.cursor()

cursor.execute(
    "INSERT INTO users (name, age) VALUES (?, ?)",
    ("Alice", 16)
)

conn.commit()
conn.close()

print("User added!")
Important
Use ? placeholders to prevent security problems.

6. Reading data (SELECT)

import sqlite3

conn = sqlite3.connect("users.db")
cursor = conn.cursor()

cursor.execute("SELECT * FROM users")

rows = cursor.fetchall()

for row in rows:
    print(row)

conn.close()

Output example:

(1, 'Alice', 16)

7. Updating and deleting data

Update

cursor.execute(
    "UPDATE users SET age = ? WHERE name = ?",
    (17, "Alice")
)

Delete

cursor.execute(
    "DELETE FROM users WHERE name = ?",
    ("Alice",)
)

Always call: 

conn.commit()

8. Putting it together — User Manager

import sqlite3

conn = sqlite3.connect("users.db")
cursor = conn.cursor()

cursor.execute("""
CREATE TABLE IF NOT EXISTS users (
    id INTEGER PRIMARY KEY,
    name TEXT,
    age INTEGER
)
""")

name = input("Enter name: ")
age = int(input("Enter age: "))

cursor.execute(
    "INSERT INTO users (name, age) VALUES (?, ?)",
    (name, age)
)

conn.commit()

cursor.execute("SELECT * FROM users")
rows = cursor.fetchall()

print("\nAll users:")
for row in rows:
    print(row)

conn.close()

Now your data is saved permanently.
