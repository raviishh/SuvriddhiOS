async and await — Writing Asynchronous Code

In this lesson, you will learn how to use async and await correctly. These two keywords allow Python to pause tasks without blocking the whole program.

1. What does async do?

When you write: 

async def task():
    print("Hello")

You are creating an asynchronous function.

Important:
Calling it does NOT run it immediately. 

task()

This only creates a coroutine object. It does not execute the function.

Simple idea
An async function is like a recipe card. You must give it to the event loop to actually cook it.

2. Running an async function

To run an async function, use: 

import asyncio

async def task():
    print("Hello")

asyncio.run(task())

asyncio.run() starts the event loop and executes the async function.

3. What does await do?

await pauses an async function until something finishes. 

import asyncio

async def task():
    print("Start")
    await asyncio.sleep(2)
    print("End")

asyncio.run(task())

await asyncio.sleep(2) waits 2 seconds. But it does NOT block the whole program.

While waiting, the event loop can run other tasks.

4. Multiple async tasks

Let’s run two async functions together. 

import asyncio

async def task_one():
    print("Task One Start")
    await asyncio.sleep(2)
    print("Task One End")

async def task_two():
    print("Task Two Start")
    await asyncio.sleep(1)
    print("Task Two End")

async def main():
    await asyncio.gather(task_one(), task_two())

asyncio.run(main())

asyncio.gather() runs multiple async tasks at the same time.

Because task_two() waits only 1 second, it finishes first.

5. time.sleep vs asyncio.sleep

Blocking sleep:

import time
time.sleep(2)

This stops the whole program.

Non-blocking sleep:

import asyncio
await asyncio.sleep(2)

This pauses only the current async task. Other async tasks can continue.

Very Important
You can only use await inside an async def function.

6. Common mistakes

  • Forgetting to use asyncio.run()
  • Using time.sleep() inside async functions
  • Calling async functions without await

If you forget await, your task may never run.

7. Small Exercise

Write a program that:

  • Creates two async functions.
  • One waits 1 second.
  • The other waits 3 seconds.
  • Run them together using asyncio.gather().

Tip
The shorter task should finish first. If not, check your await statements.
