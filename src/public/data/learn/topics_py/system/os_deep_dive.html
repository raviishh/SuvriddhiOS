OS Module Deep Dive (Python)

Learn how Python talks to your computer’s operating system. In this lesson, you will control folders, check system details, and work with file paths safely.

1. What is an Operating System?

The operating system (OS) is the main software that controls your computer. Windows, macOS, and Linux are operating systems.

The OS manages:

  • Files and folders
  • Programs
  • Memory
  • Hardware (keyboard, screen, etc.)

Analogy
The OS is like a school principal. Programs (like Python) must ask the principal before using rooms, files, or equipment.

Python uses a built-in module called os to talk to the operating system.

2. Importing the os Module

To use it, we first import it: 

import os

Now Python can:

  • Check the current folder
  • List files
  • Create folders
  • Remove folders
  • Get system information

3. Working with Directories

Check Current Folder

import os

print(os.getcwd())

getcwd() means "get current working directory". It shows the folder where your program is running.

List Files in Folder

import os

files = os.listdir()
print(files)

listdir() shows all files and folders inside the current directory.

Change Folder

import os

os.chdir("my_folder")
print(os.getcwd())

chdir() means "change directory".

4. Creating and Removing Folders

Create a Folder

import os

os.mkdir("test_folder")

This creates a new folder called test_folder.

Remove a Folder

import os

os.rmdir("test_folder")

This removes the folder. The folder must be empty, or it will give an error.

5. Working with File Paths (Very Important)

A file path is the location of a file. There are two types:

  • Absolute path – Full location from the root
  • Relative path – Location from current folder

Important
Different systems use different slashes. Windows uses "\" and Linux/macOS use "/".

To avoid problems, use os.path.join(). 

import os

folder = "my_folder"
file = "data.txt"

path = os.path.join(folder, file)
print(path)

This safely builds a correct path for your system.

6. Getting System Information

Check Operating System Type

import os

print(os.name)

It may print:

  • nt (Windows)
  • posix (Linux/macOS)

Check CPU Count

import os

print(os.cpu_count())

This shows how many CPU cores your computer has.
