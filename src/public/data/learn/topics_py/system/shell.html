Working with the Shell (Python)

Learn how to use the terminal (shell) and how Python works with it. In this lesson, you will run programs from the command line and pass information into Python.

1. What Is the Shell?

The shell (also called the terminal or command line) is a text-based way to control your computer.

Instead of clicking with a mouse, you type commands.

Analogy
The shell is like a remote control for your computer. You type instructions, and the computer obeys.

You can:

  • Move between folders
  • Create or delete files
  • Run programs

2. Basic Shell Commands

Here are common commands:

  • pwd – Show current folder (Linux/macOS)
  • ls – List files (Linux/macOS)
  • cd – Change folder
  • mkdir – Create folder
  • rm – Delete file (Linux/macOS)

On Windows:

  • dir instead of ls
  • del instead of rm

These commands are not Python. They are part of your operating system.

3. Running Python from the Shell

If you have a file called program.py, you can run it like this: 

python program.py

The shell tells Python to start the program.

You can also start interactive mode: 

python

Then you can type Python code directly.

4. Passing Arguments to a Python Program

Sometimes we want to give information to a program from the shell. 

python program.py Alice

Here, Alice is an argument.

Inside Python, we use the sys module to read arguments. 

import sys

print(sys.argv)

sys.argv is a list:

  • The first item is the file name
  • The next items are arguments

Example

import sys

if len(sys.argv) > 1:
    print("Hello,", sys.argv[1])
else:
    print("Hello, Guest")

If you run: 

python program.py Bob

Output:

Hello, Bob

5. Small Practice Project — Command-Line Calculator

Create a simple calculator that adds two numbers from the shell. 

import sys

if len(sys.argv) == 3:
    a = int(sys.argv[1])
    b = int(sys.argv[2])
    print("Result:", a + b)
else:
    print("Usage: python program.py number1 number2")

Run it like this: 

python program.py 5 7
Result: 12
Friendly Tip
Many real programs use command-line arguments. This makes your programs flexible and powerful.
