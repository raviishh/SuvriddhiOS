Running Programs with subprocess (Python)

Learn how Python can run other programs and system commands. In this lesson, Python becomes a manager that tells other programs what to do.

1. Why Run Other Programs?

Sometimes your Python program needs to:

  • Run a system command (like ls or dir)
  • Run another Python file
  • Use tools already installed on your computer

Analogy
Think of Python as a manager. It can hire another worker (program) to do a job and then read the result.

Python uses a built-in module called subprocess for this.

2. Importing subprocess

import subprocess

Now Python can run system commands.

3. Running a Simple Command

import subprocess

subprocess.run(["ls"])

This runs the ls command (used in Linux/macOS to list files).

On Windows, use: 

subprocess.run(["dir"], shell=True)

We use a list like ["ls"] because each word in a command should be a separate item in the list.

4. Capturing Output

Sometimes we want to save the result instead of just printing it. 

import subprocess

result = subprocess.run(
    ["ls"],
    capture_output=True,
    text=True
)

print(result.stdout)

Explanation:

  • capture_output=True saves the output
  • text=True gives output as normal text
  • result.stdout contains the result

5. Running Another Python File

You can run another Python script from your program. 

import subprocess

subprocess.run(["python", "otherfile.py"])

This runs otherfile.py just like typing it in the terminal.

6. Handling Errors

If a command fails, we can make Python raise an error. 

import subprocess

subprocess.run(["ls", "unknown_folder"], check=True)

check=True makes Python show an error if the command fails.

Programs return a number called a "return code":

  • 0 means success
  • Other numbers mean something went wrong

7. Small Practice Challenge

Try writing a program that:

  • Runs a command to list files
  • Saves the output
  • Prints the output

Friendly Tip
If your command does not work, try running it directly in the terminal first. Make sure it works there.
