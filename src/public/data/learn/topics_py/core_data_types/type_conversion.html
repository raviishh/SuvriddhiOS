Type Conversion

Sometimes you need to change one data type into another. A number might arrive as text. A Boolean might need to become a number. This process is called type conversion.

1. Why type conversion matters

Different data types behave differently. You can’t safely mix them without understanding what Python will do. 

age = "18"
print(age + 1)   # ❌ TypeError

Python prevents you from accidentally mixing incompatible types. You must convert explicitly.

2. Converting to integers

Use int() to convert values into integers. 

age = "18"
age_num = int(age)

print(age_num + 1)  # 19

You can also convert floats: 

print(int(3.9))   # 3 (decimal is removed)
Important
int() does NOT round — it truncates (cuts off the decimal part).

3. Converting to floats

Use float() to create floating-point numbers. 

print(float("3.14"))  # 3.14
print(float(5))       # 5.0

This is useful when working with user input or division results.

4. Converting to strings

Use str() to convert values into text. 

score = 95
message = "Your score is " + str(score)

print(message)

This is especially useful when combining numbers with strings.

5. Converting to Booleans

Use bool() to convert values into True or False. 

print(bool(0))        # False
print(bool(1))        # True
print(bool(""))       # False
print(bool("Hello"))  # True

Rule of thumb:

  • Zero and empty values → False
  • Non-zero and non-empty → True

6. Invalid conversions

Not every conversion is possible. 

print(int("hello"))  # ❌ ValueError

Python raises an error when conversion doesn’t make logical sense. Always validate input if you’re unsure.

7. Implicit vs explicit conversion

Python sometimes converts types automatically (implicit conversion). 

print(5 + 2.0)  # 7.0 (int automatically becomes float)

But in most cases, you must convert manually (explicit conversion). Explicit is better — it keeps your intent clear.

8. Common beginner mistakes

  • Forgetting to convert input() (which always returns a string)
  • Assuming int() rounds instead of truncates
  • Trying to convert non-numeric text to numbers
  • Over-relying on implicit conversion
Quick Check
What happens if you run: int("10.5")? Why?

9. Where this fits in Core Data Types

Type conversion connects all core data types together. Without it, numbers, strings, and Booleans would remain isolated.
