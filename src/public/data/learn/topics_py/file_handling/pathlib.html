Pathlib — The Modern Way to Handle File Paths

String-based path manipulation is fragile. pathlib fixes that. It treats file paths as objects — not raw text — making your code cleaner, safer, and more expressive.

1. Why pathlib exists

Historically, Python developers used os.path for path operations. It works, but it’s procedural and string-heavy.

pathlib introduces an object-oriented approach: paths become objects with methods.

Design Philosophy
Treat paths as structured data — not strings you manually glue together.

2. Creating Path objects

from pathlib import Path

path = Path("data.txt")
print(path)

The Path object represents a filesystem path.

Building paths becomes intuitive: 

folder = Path("reports")
file = folder / "summary.txt"

print(file)

The / operator joins paths correctly across operating systems.

3. Checking existence and type

path = Path("reports")

if path.exists():
    print("Exists")

if path.is_file():
    print("It is a file")

if path.is_dir():
    print("It is a directory")

No need for os.path.exists(). Cleaner. Clearer.

4. Creating directories

folder = Path("reports")

folder.mkdir(exist_ok=True)

exist_ok=True prevents errors if the folder already exists.

For nested directories: 

folder.mkdir(parents=True, exist_ok=True)

5. Reading and writing files with Path

Path includes convenience methods. 

file = Path("example.txt")

# Write text
file.write_text("Hello, Pathlib!\n")

# Read text
content = file.read_text()
print(content)

These methods automatically open and close files internally.

Efficiency Note
For large files or advanced control (encoding, binary mode), you may still prefer traditional with open().

6. Getting absolute paths

file = Path("example.txt")

print(file.resolve())

resolve() returns the absolute path.

7. Iterating over directory contents

folder = Path("reports")

for item in folder.iterdir():
    print(item)

Filtering by extension: 

for file in folder.glob("*.txt"):
    print(file)

8. Why modern code prefers pathlib

You now understand:

  • How paths become objects
  • How to build cross-platform paths safely
  • How to read/write directly from Path objects
  • How to iterate and filter files cleanly

pathlib improves readability and reduces bugs caused by manual string handling. It represents the modern, Pythonic way to manage filesystem operations.
