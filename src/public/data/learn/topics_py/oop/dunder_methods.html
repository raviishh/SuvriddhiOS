Dunder Methods

Dunder methods are special methods in Python. They have double underscores before and after the name. Example: __init__.

“Dunder” means “double underscore.”

1. What Are Dunder Methods?

Dunder methods allow us to control how objects behave.

They are also called magic methods.

Simple Idea
Dunder methods tell Python what to do when we use built-in functions like print(), len(), or even +.

2. The __str__ Method

The __str__ method controls what happens when we print an object.

Without __str__: 

class Book:
    def __init__(self, title):
        self.title = title

book = Book("Python Basics")
print(book)

The output will look strange, like a memory address.

Now let us improve it. 

class Book:
    def __init__(self, title):
        self.title = title

    def __str__(self):
        return "Book: " + self.title

book = Book("Python Basics")
print(book)

Now the output is clean and readable.

3. The __len__ Method

The __len__ method allows us to use len() on an object. 

class Playlist:
    def __init__(self, songs):
        self.songs = songs

    def __len__(self):
        return len(self.songs)

playlist = Playlist(["Song1", "Song2", "Song3"])
print(len(playlist))

Explanation:

  • Python calls __len__ when we use len().
  • We control what length means for our object.

4. Operator Overloading (__add__)

Dunder methods also control operators like +. 

class Number:
    def __init__(self, value):
        self.value = value

    def __add__(self, other):
        return Number(self.value + other.value)

    def __str__(self):
        return str(self.value)

n1 = Number(5)
n2 = Number(3)

result = n1 + n2
print(result)

Explanation:

  • __add__ runs when we use +.
  • We define what addition means for this class.

This is called operator overloading.

5. Why Use Dunder Methods?

Dunder methods:

  • Make objects behave like built-in types
  • Improve readability
  • Give full control over object behavior

Important
Do not create your own random dunder names. Only use the special ones defined by Python.

6. Small Exercise

Create a class called Box.

  • Add an attribute items (a list).
  • Implement __len__ so len(box) returns number of items.
  • Implement __str__ so printing the box shows the items clearly.

7. What You Learned

You now understand:

  • Dunder methods are special Python methods.
  • __str__ controls printing.
  • __len__ controls length.
  • __add__ controls the + operator.

Friendly Tip
Dunder methods make your classes feel powerful. But use them carefully. Only override behavior when it makes logical sense.
