Methods & self

In this lesson, you will learn how objects can perform actions. You will also understand the meaning of self.

1. What is a Method?

A method is a function inside a class.

If attributes are data, methods are actions.

Analogy
A phone has data like brand and battery level. But it can also call and send messages. Those actions are like methods. 
class Dog:
    def bark(self):
        print("Woof!")

Explanation:

  • def bark(self): defines a method.
  • Methods are written inside a class.

2. Calling a Method

To use a method, we create an object and call it. 

class Dog:
    def bark(self):
        print("Woof!")

dog1 = Dog()
dog1.bark()

Explanation:

  • dog1.bark() calls the method.
  • The object performs the action.

The output will be:

Woof!

3. What is "self"?

The word self means “this object.”

It allows the method to access the object’s data. 

class Dog:
    def set_name(self, name):
        self.name = name

    def speak(self):
        print("My name is", self.name)

dog1 = Dog()
dog1.set_name("Buddy")
dog1.speak()

Explanation:

  • self.name stores data inside the object.
  • Each object has its own self.
  • Python automatically sends the object as the first argument.

If we create another dog: 

dog2 = Dog()
dog2.set_name("Rocky")
dog2.speak()

Each dog remembers its own name.

4. Why Must We Write self?

When a method runs, it needs to know: “Which object is using me?”

The self parameter answers that question.

Important Rule
Inside a class, every method must have self as the first parameter. It can have other parameters too.

Example with more parameters: 

class Calculator:
    def add(self, a, b):
        return a + b

calc = Calculator()
print(calc.add(3, 4))

Even though we wrote calc.add(3, 4), Python secretly calls it like this: 

Calculator.add(calc, 3, 4)

The object is passed automatically.

Create a class called Person.

  • Add a method set_age(self, age).
  • Add a method show_age(self).
  • Create two objects with different ages.
  • Call the methods and print the results.

6. What You Learned

You now understand:

  • A method is a function inside a class.
  • Methods define actions for objects.
  • self refers to the current object.
  • Each object keeps its own data.

Friendly Tip
If you forget to write self, Python will give an error. When that happens, check your method definition first.
