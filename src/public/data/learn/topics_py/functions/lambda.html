Lambda Functions in Python

Lambda functions are small, anonymous functions defined in a single line. They are used for short, simple operations — especially when passing functions as arguments to other functions.

1. What Is a Lambda Function?

A lambda function is created using the lambda keyword. It has no name unless assigned to a variable. 

square = lambda x: x * x
print(square(4))

This behaves similarly to: 

def square(x):
    return x * x
Key Difference
Lambda functions are limited to a single expression. They cannot contain multiple statements.

2. Basic Syntax

lambda parameters: expression
  • No def keyword
  • No function name (unless assigned)
  • No explicit return
  • Automatically returns the expression value

3. Multiple Parameters

add = lambda a, b: a + b
print(add(3, 5))

Lambdas can accept multiple parameters, just like regular functions.

4. Common Use Cases

With map()

numbers = [1, 2, 3, 4]
squared = list(map(lambda x: x * x, numbers))
print(squared)

With filter()

evens = list(filter(lambda x: x % 2 == 0, numbers))
print(evens)

With sorted()

pairs = [(1, 3), (2, 1), (4, 2)]
sorted_pairs = sorted(pairs, key=lambda x: x[1])
print(sorted_pairs)

Lambdas are commonly used when a short function is needed temporarily.

5. When Not to Use Lambda

Avoid lambdas when:

  • The logic is complex
  • You need multiple statements
  • Readability suffers
  • The function will be reused in multiple places
Design Guideline
If the function needs a comment to explain it, consider using a regular def.

6. Limitations

  • Single expression only
  • No statements (e.g., loops, assignments)
  • No annotations
  • Can reduce readability if overused

7. Common Mistakes

  • Using lambda for complex logic
  • Assigning lambda when def would be clearer
  • Forgetting that lambda automatically returns its expression
Quick Check
What prints? f = lambda x: x + 2 print(f(5))

8. Why Lambda Functions Matter

Lambda functions enable concise functional-style programming. They are powerful tools for transforming data streams and customizing behavior dynamically.

Used appropriately, they enhance clarity. Overused, they obscure intent.

Practice Strategy
Replace a simple one-line helper function with a lambda in a map() or sorted() call. Compare readability.
