Regular Expressions (re)

In this lesson, you will learn how to search for patterns inside text. This is done using regular expressions.

1. What is a regular expression?

A regular expression (regex) is a pattern. It describes what text should look like.

Simple Analogy
Think of regex like a search rule. Instead of searching for one exact word, you search for a pattern.

For example:

  • Find all numbers in a sentence.
  • Find email addresses.
  • Find words that start with capital letters.

2. Using the re module

Python provides regular expressions through the re module. 

import re

text = "My age is 18"

match = re.search(r"\d+", text)

print(match.group())

Output:

18

Explanation:

  • \d means a digit.
  • + means one or more.
  • r"" means raw string (important for regex).

3. Common regex patterns

  • . → any character
  • \d → digit (0–9)
  • \w → word character (letters, numbers, _)
  • \s → space
  • * → zero or more
  • + → one or more
import re

text = "cat bat rat"

words = re.findall(r"\w+at", text)
print(words)

Output:

['cat', 'bat', 'rat']

4. search() vs findall()

  • search() → finds first match
  • findall() → finds all matches

import re

text = "There are 3 apples and 5 bananas"

first_number = re.search(r"\d+", text)
all_numbers = re.findall(r"\d+", text)

print(first_number.group())
print(all_numbers)

Output:

3 ['3', '5']

5. Simple email example

Let us find email-like text. 

import re

text = "Contact me at test@example.com"

pattern = r"\w+@\w+\.\w+"

match = re.search(pattern, text)

print(match.group())

Output:

test@example.com

This pattern means:

  • Word characters
  • @ symbol
  • Word characters
  • Dot
  • Word characters

6. Replacing text

We can also replace matched text using sub(). 

import re

text = "My number is 12345"

new_text = re.sub(r"\d+", "XXXXX", text)

print(new_text)

Output:

My number is XXXXX

7. Why regular expressions matter

Regular expressions are useful for:

  • Validating input
  • Cleaning data
  • Searching large text files

Important Idea
Regex is powerful. But start simple. Complex patterns can become hard to read.
