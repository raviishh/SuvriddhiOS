Input & Output in Python

A clear and friendly guide to how Python talks to the outside world — taking input from users and showing output on the screen. Think of this as teaching your program how to listen and speak.

1. What is Input and Output?

Programs are not very useful if they only think silently. They need to communicate. Input is information the program receives. Output is information the program sends out.

Analogy
Imagine Python is a helpful assistant. Input is you giving instructions. Output is the assistant replying with results.

In Python, the most basic tools for this are:

  • print() — for output
  • input() — for input

2. Output — using print()

The print() function displays information on the screen. 

print("Hello, World!")
print(5)
print("The answer is", 42)

Explanation:

  • Text goes inside quotes.
  • Numbers don’t need quotes.
  • You can print multiple things separated by commas.

Try it: Run this in a Python file or interactive shell.
Output:
Hello, World! 5 The answer is 42

3. Input — using input()

The input() function waits for the user to type something and press Enter. 

name = input("Enter your name: ")
print("Hello,", name)

How it works:

  • The text inside input() is a prompt shown to the user.
  • Whatever the user types is stored in a variable.
  • input() always returns a string.

Important Note
Even if the user types a number, Python stores it as text unless you convert it.

4. Converting Input — strings to numbers

Since input() gives you text, you must convert it if you want numbers. 

age = input("Enter your age: ")
age = int(age)

print("Next year you will be", age + 1)

Cleaner version: 

age = int(input("Enter your age: "))
print("Next year you will be", age + 1)

Common conversions:

  • int() → whole numbers
  • float() → decimals
  • str() → convert to text

5. Formatting Output — making it neat

Sometimes you want cleaner, more controlled output. Python’s f-strings help with that. 

name = "Aisha"
score = 95

print(f"{name} scored {score} marks.")

Why use f-strings?

  • They’re readable.
  • You can insert variables directly inside curly braces { }.

Tip
Always prefer f-strings over older formatting methods. They are cleaner and modern.

6. Simple Practice Program

Let’s combine input and output into a tiny interactive program. 

num1 = float(input("Enter first number: "))
num2 = float(input("Enter second number: "))

sum_result = num1 + num2

print(f"The sum is {sum_result}")

This program:

  • Takes two numbers as input
  • Converts them to floats
  • Prints their sum
