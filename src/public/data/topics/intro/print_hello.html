Your First C Program: Printing “Hello, World!”

Welcome to your very first C exercise! Today, you're going to make your computer talk back to you — well, at least on the screen. This is a classic step for every programmer: writing the Hello, World! program.

What We're Doing

When you run a C program, your computer follows the instructions you give it, step by step. Right now, our only instruction will be: “Hey computer, show the message Hello, World! on the screen.”

Here's the Code


#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}

Let's break it down line by line:

  • #include <stdio.h> — this is like saying, “Please give me access to the tools that let me print text.” The file stdio.h contains useful input/output functions.
  • int main() — this is the main function. Every C program starts running from here.
  • printf("Hello, World!\n"); — this prints the message onto the screen. The \n at the end tells your computer to go to a new line afterwa
