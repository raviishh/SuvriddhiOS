Atomic Structure

Part 1 — The Nucleus and Subatomic Particles

The Atom Isn't What Dalton Thought

John Dalton proposed in 1803 that atoms were indivisible solid spheres — the ultimate building block of matter. That turned out to be wrong. By the early 20th century, experiments had shown that atoms have internal structure: a tiny, dense, positively charged nucleus surrounded by negatively charged electrons in a vast amount of empty space.

If an atom were scaled up to the size of a football stadium, the nucleus would be a marble at the centre. Everything else would be empty space, with electrons somewhere in the stands.

The Three Subatomic Particles

Protons

Charge: +1    Mass: 1 amu    Location: Nucleus

The number of protons defines what element an atom is. All carbon atoms have 6 protons — always. If you change the proton count, you have a different element. This number is called the atomic number (Z).

Neutrons

Charge: 0    Mass: 1 amu    Location: Nucleus

Neutrons have no charge. They contribute to the atom's mass and help hold the nucleus together (nuclear strong force). Because they're neutral, varying neutron count doesn't change the element's chemical identity — but it does change the mass number.

Electrons

Charge: −1    Mass: ~1/1836 amu (negligible)    Location: Shells around nucleus

Electrons are tiny — their mass is so small it's ignored in most calculations. In a neutral atom, the number of electrons equals the number of protons, so charges cancel. Electrons in the outer shell (valence electrons) are responsible for all chemical bonding and reactivity.

Atomic Number (Z) = number of protons

Mass Number (A) = protons + neutrons

Neutrons = A − Z

In a neutral atom: electrons = protons = Z

Example: Chlorine-35

Chlorine has Z = 17, A = 35.

Protons = 17, Neutrons = 35 − 17 = 18, Electrons = 17

Written in nuclide notation: ³⁵₁₇Cl (mass number on top, atomic number on bottom)

The subscript (17) is actually redundant — if you know the symbol is Cl, you know Z = 17. But it's useful for clarity in nuclear equations.

Isotopes

Atoms of the same element always have the same number of protons — but they can have different numbers of neutrons. These variants are called isotopes.

Isotopes are atoms of the same element with the same atomic number but different mass numbers (different neutron counts).

They have identical chemical properties (same electron configuration, same valence electrons) but slightly different physical properties (different mass, density, rate of diffusion).

Relative Atomic Mass

Because most elements exist as a mix of isotopes, the atomic mass shown on the periodic table is a weighted average of all naturally occurring isotopes, weighted by their abundance.

Relative Atomic Mass (Aᵣ) = Σ (isotope mass × fractional abundance)

Example: Relative Atomic Mass of Chlorine

Chlorine exists as two isotopes:

³⁵Cl: mass = 35, abundance = 75.77%

³⁷Cl: mass = 37, abundance = 24.23%

Aᵣ = (35 × 0.7577) + (37 × 0.2423) = 26.52 + 8.97 = 35.5

This is why chlorine's mass on the periodic table is 35.5 — not a whole number. It's a weighted average, not an actual atom.

Ions

Neutral atoms have equal numbers of protons and electrons. If an atom gains or loses electrons, it becomes a charged particle — an ion.

Cation (+): Loses electrons. Protons > electrons. Metals typically form cations.
Na (11p, 11e) → Na⁺ (11p, 10e)

Anion (−): Gains electrons. Electrons > protons. Non-metals typically form anions.
Cl (17p, 17e) → Cl⁻ (17p, 18e)

The number of protons never changes when forming an ion — only electrons are gained or lost. Changing the proton count would mean becoming a different element entirely (which requires nuclear reactions, not chemistry).
