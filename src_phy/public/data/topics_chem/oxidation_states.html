Redox Reactions

Part 2 — Oxidation States

Tracking Electrons with Oxidation States

For simple reactions like Zn + Cu²⁺, it's obvious who loses and gains electrons. But for reactions involving covalent compounds — where electrons are shared, not transferred — we need a bookkeeping system. Oxidation states (also called oxidation numbers) are a formal way to track electron distribution, even in covalent molecules.

Rules for Assigning Oxidation States

Apply in this priority order:

1. Pure element: oxidation state = 0   (Na, O₂, Fe, Cl₂ → all 0)

2. Simple monatomic ion: = the ion charge   (Na⁺ = +1, Cl⁻ = −1, Fe³⁺ = +3)

3. Fluorine: always −1 in compounds

4. Oxygen: −2 in most compounds (except peroxides: −1; OF₂: +2)

5. Hydrogen: +1 in compounds with non-metals; −1 in metal hydrides (NaH)

6. Sum of all oxidation states in a neutral compound = 0

7. Sum in a polyatomic ion = the ion's charge

Using Oxidation States to Identify Redox

In a redox reaction, the oxidation state of at least one element changes. If no oxidation states change, it's not a redox reaction.

Increase in oxidation state = Oxidation (electrons lost)

Decrease in oxidation state = Reduction (electrons gained)

Memory trick: "LEO the lion says GER" — Loss Electrons Oxidation / Gain Electrons Reduction. Same as OIL RIG, just another way to remember.

Example: Is This Redox?

Reaction 1: 2Mg + O₂ → 2MgO

Mg: 0 → +2 (oxidized).   O: 0 → −2 (reduced).   ✓ Redox

Reaction 2: NaOH + HCl → NaCl + H₂O

Na: +1 → +1.   O: −2 → −2.   H: +1 → +1.   Cl: −1 → −1.

No oxidation state changes. ✗ NOT redox — it's a simple acid-base neutralization.

Checking oxidation states is the definitive test for whether a reaction is redox. Don't guess based on the type of substances involved.

Variable Oxidation States — Transition Metals

Unlike main group elements, transition metals can take multiple oxidation states. This is one of their defining features.

Iron: +2 (Fe²⁺, ferrous) and +3 (Fe³⁺, ferric) — both common in compounds.

Copper: +1 (Cu⁺) and +2 (Cu²⁺) — CuO is copper(II) oxide.

Manganese: +2, +4, +7 — the MnO₄⁻ ion has Mn at +7.

Chromium: +2, +3, +6 — Cr₂O₇²⁻ has Cr at +6.

Roman numerals in compound names tell you the oxidation state: iron(III) chloride = FeCl₃ (Fe is +3). This avoids ambiguity when an element has multiple states.

Disproportionation: Some elements can simultaneously be oxidized and reduced in the same reaction — acting as both oxidizing and reducing agent. Example: 2H₂O₂ → 2H₂O + O₂. Oxygen goes from −1 (in H₂O₂) to −2 (in H₂O) and 0 (in O₂) simultaneously.
