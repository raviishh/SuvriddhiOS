Laboratory Techniques

Part 1 — Titration

What Is Titration?

Titration is a quantitative technique for finding the concentration of an unknown solution by reacting it with a standard solution (known concentration) until the reaction is exactly complete. The point at which stoichiometrically equivalent amounts have reacted is called the equivalence point.

Standard solution: A solution of precisely known concentration, used as the reference.

Titre: The volume of standard solution needed to reach the end point.

End point: The point detected experimentally (e.g. indicator colour change). Ideally coincides with the equivalence point.

The Titration Procedure

1. Rinse burette with standard solution; fill and record initial reading (to 2 decimal places, e.g. 0.00 cm³)

2. Pipette a fixed volume of unknown solution into conical flask (e.g. 25.00 cm³ using volumetric pipette)

3. Add 2–3 drops of appropriate indicator

4. Add standard solution slowly from burette, swirling continuously

5. Near end point, add dropwise — a single drop changes the colour permanently

6. Record final burette reading; titre = final − initial

7. Repeat until concordant results (within 0.10 cm³ of each other)

8. Average the concordant titres

Why concordant results? A single reading may have a random error (air bubble in tip, overshoot). Concordant results show reproducibility and allow random errors to be averaged out. Systematic errors (wrong indicator, calibration) would still affect all readings equally.

Choosing the Right Indicator

Strong acid + Strong base (e.g. HCl + NaOH): pH jumps sharply at equivalence point (pH 3→11). Either phenolphthalein or methyl orange works.

Strong acid + Weak base (e.g. HCl + NH₃): equivalence point ~pH 5. Use methyl orange (changes 3.1–4.4).

Weak acid + Strong base (e.g. CH₃COOH + NaOH): equivalence point ~pH 9. Use phenolphthalein (changes 8.2–10).

Weak acid + Weak base: No sharp end point — cannot be titrated reliably with a simple indicator.

Titration Calculation

n = c × V   (V in dm³)

Use mole ratio from balanced equation to link moles of standard → moles of unknown.

Then c(unknown) = n / V(unknown)

Example: Finding Concentration of Sulfuric Acid

25.0 cm³ of H₂SO₄ is titrated with 0.100 mol/dm³ NaOH. Average titre = 23.60 cm³.

Equation: H₂SO₄ + 2NaOH → Na₂SO₄ + 2H₂O

n(NaOH) = 0.100 × 0.02360 = 2.360 × 10⁻³ mol

n(H₂SO₄) = 2.360×10⁻³ / 2 = 1.180 × 10⁻³ mol   (ratio 1:2)

c(H₂SO₄) = 1.180×10⁻³ / 0.0250 = 0.0472 mol/dm³

Note the 1:2 ratio — this is the most common mark-losing step. Always write the equation and identify the mole ratio before calculating.

Back Titration

Used when the substance being analyzed doesn't react directly or cleanly with a standard solution. An excess of reagent is added, and the leftover is titrated.

Example: Finding % CaCO₃ in limestone.

Add excess HCl (known moles) to the limestone. CaCO₃ reacts, leaving unused HCl.

Titrate the excess HCl with NaOH → find moles of excess HCl.

Moles CaCO₃ = moles HCl added − moles HCl excess.
