Sequences

What is a Sequence?

A sequence is an ordered list of numbers following a particular pattern. Each number in the sequence is called a term. The nth term is written as an or Tn.

Arithmetic Progressions (AP)

In an arithmetic progression, each term is obtained by adding a fixed value called the common difference (d) to the previous term.

Example: 3, 7, 11, 15, ... → common difference d = 4

First term = a, Common difference = d

nth Term of an AP:

Tn = a + (n - 1)d

Sum of first n terms of an AP:

Sn = n/2 × [2a + (n-1)d]

Or equivalently: Sn = n/2 × (first term + last term)

Example

Find the 10th term and sum of first 10 terms of 5, 8, 11, 14, ...

a = 5, d = 3

T10 = 5 + 9 × 3 = 32

S10 = 10/2 × [10 + 32] = 5 × 42 = 210

Geometric Progressions (GP)

In a geometric progression, each term is obtained by multiplying by a fixed value called the common ratio (r).

Example: 2, 6, 18, 54, ... → common ratio r = 3

First term = a, Common ratio = r

nth Term of a GP:

Tn = a × r(n-1)

Sum of first n terms of a GP:

Sn = a(rn - 1) / (r - 1)    [for r ≠ 1]

If r = 1, then Sn = na

Example

Find the 5th term and sum of first 5 terms of 3, 6, 12, 24, ...

a = 3, r = 2

T5 = 3 × 24 = 3 × 16 = 48

S5 = 3(25 - 1)/(2 - 1) = 3 × 31 = 93

Infinite Geometric Series

If the common ratio satisfies |r| < 1, the terms of a GP get smaller and smaller and the sum converges to a finite value.

Sum to Infinity (|r| < 1):

S∞ = a / (1 - r)

Example

Find the sum to infinity of 8, 4, 2, 1, ...

a = 8, r = 0.5

S∞ = 8 / (1 - 0.5) = 16

Why does it converge? Each term is half the previous one, so the terms shrink toward zero. Adding infinitely many increasingly tiny numbers still gives a finite total.

AP vs GP: Quick Comparison

  • AP: add/subtract a fixed number each time | linear growth
  • GP: multiply/divide by a fixed number each time | exponential growth or decay
  • To check AP: differences between consecutive terms are equal
  • To check GP: ratios between consecutive terms are equal
