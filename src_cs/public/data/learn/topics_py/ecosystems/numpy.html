NumPy Basics

A simple introduction to NumPy — the fast and powerful math library in Python. Think of NumPy as a super calculator that works with many numbers at the same time.

1. What is NumPy?

NumPy is a Python library used for working with numbers. It is very fast and is used in science, data analysis, and artificial intelligence.

Analogy
A normal Python list is like writing numbers in a notebook. NumPy is like using a scientific calculator — faster and built for math.

Many other libraries (like Pandas and machine learning tools) use NumPy inside them. So learning NumPy is very important.

2. Installing and Importing NumPy

First, install NumPy using pip: 

pip install numpy

Then import it in Python: 

import numpy as np

We write as np because it is shorter and easier to type.

3. Creating NumPy Arrays

NumPy works with something called an array. An array is like a list, but faster and more powerful. 

import numpy as np

numbers = np.array([1, 2, 3, 4])
print(numbers)

Output:

[1 2 3 4]

You can also create 2D arrays (like tables): 

matrix = np.array([[1, 2],
                   [3, 4]])
print(matrix)

This is called a 2D array. It has rows and columns.

Tip
1D array = one line of numbers.
2D array = table of numbers.

4. Special Arrays

NumPy can quickly create arrays filled with zeros or ones. 

zeros = np.zeros(3)
print(zeros)

ones = np.ones(4)
print(ones)

You can also create a range of numbers: 

numbers = np.arange(0, 5)
print(numbers)

This creates numbers from 0 to 4.

5. Fast Math with Arrays

NumPy can do math on all numbers at the same time. You do not need loops. 

import numpy as np

a = np.array([1, 2, 3])
b = np.array([4, 5, 6])

print(a + b)
print(a * 2)

Output:

[5 7 9]
[2 4 6]

This is called element-wise operation. Each number is calculated one by one automatically.

6. Shape and Size

You can check the shape of an array. Shape tells you how many rows and columns it has. 

matrix = np.array([[1, 2, 3],
                   [4, 5, 6]])

print(matrix.shape)

Output:

(2, 3)

This means 2 rows and 3 columns.

7. Why NumPy Matters

NumPy is used in:

  • Data science
  • Artificial intelligence
  • Scientific research
  • Game engines

It is the foundation for many advanced Python tools. If you understand NumPy, you understand how many data tools work.
