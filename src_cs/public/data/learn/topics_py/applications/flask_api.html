Simple REST API with Flask

In this lesson, you will learn how to build a small web server using Flask. By the end, you will create your own Notes API that sends and receives data using JSON.

1. What is a server?

A server is a program that waits for requests and sends back responses.

Simple example
Your browser asks: “Give me a webpage.”
The server replies: “Here it is.”

When you open a website:

  • Your browser sends a request
  • A server receives it
  • The server sends back data

Today, you will build your own server.

2. What is an API?

API means Application Programming Interface.

A REST API is a server that:

  • Receives requests
  • Returns data (usually JSON)
Think of it like this
An API is like a waiter in a restaurant.
You ask for food. The waiter brings it.

3. Installing Flask

Flask is a small Python library for building web servers.

Install Flask:
pip install flask

Now you are ready to build your first server.

4. Your first Flask server

from flask import Flask

app = Flask(__name__)

@app.route("/")
def home():
    return "Hello from my server!"

if __name__ == "__main__":
    app.run(debug=True)

Save as app.py and run:

python app.py

Then open your browser and go to:

http://127.0.0.1:5000/

You just built a working web server.

5. Understanding routes

A route tells Flask what to do when someone visits a URL.

@app.route("/hello")
def hello():
    return "Hello there!"

Visiting:

http://127.0.0.1:5000/hello

will show:

Hello there!

6. Returning JSON data

Most APIs return JSON.

from flask import Flask, jsonify

app = Flask(__name__)

@app.route("/notes")
def get_notes():
    notes = ["Buy milk", "Study math"]
    return jsonify(notes)

When you visit:

http://127.0.0.1:5000/notes

You will see:

["Buy milk", "Study math"]

That is JSON data.

7. Handling POST requests

GET is used to read data. POST is used to send data.

from flask import Flask, request, jsonify

app = Flask(__name__)
notes = []

@app.route("/notes", methods=["POST"])
def add_note():
    data = request.get_json()
    note = data.get("note")
    notes.append(note)
    return jsonify({"message": "Note added!"})

To test POST requests, you can use:

  • Postman
  • curl

8. Putting it together — Notes API

from flask import Flask, request, jsonify

app = Flask(__name__)
notes = []

@app.route("/notes", methods=["GET"])
def get_notes():
    return jsonify(notes)

@app.route("/notes", methods=["POST"])
def add_note():
    data = request.get_json()
    note = data.get("note")
    notes.append(note)
    return jsonify({"message": "Note added!"})

if __name__ == "__main__":
    app.run(debug=True)

Your API now:

  • Stores notes in memory
  • Returns them as JSON
Important
The data will reset when the server stops. In a later lesson, you will store data in a database.
