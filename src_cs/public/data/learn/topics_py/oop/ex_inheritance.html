Inheritance Hierarchy Design

Build a shape hierarchy using inheritance and method overriding.

Part 1 — Circle and Rectangle

A subclass inherits all methods from its parent. Override a method by redefining it in the subclass.

class Animal:
    def speak(self):
        return "..."

class Dog(Animal):
    def speak(self):
        return "Woof!"

print(Dog().speak())  # Woof!
Output:
Woof!

What to do

  • Create Circle(Shape) with radius, overriding area() to return math.pi * radius ** 2.
  • Create Rectangle(Shape) with width and height, overriding area() to return width * height.
  • Print describe() for Circle(5) and Rectangle(4, 6).

Watch out for

  • area is formatted to 2 decimal places inside describe() — do not reformat it.
  • Do not override describe() — the whole point is that it is inherited and still works correctly.
Hints (4)
  • class Circle(Shape): def __init__(self, radius): self.radius = radius
  • def area(self): return math.pi * self.radius ** 2
  • describe() is already defined on Shape — you get it for free.
  • type(self).__name__ returns the class name as a string — 'Circle' or 'Rectangle'.

Stretch — Triangle and Polymorphic Area Sum stretch

Polymorphism lets you treat objects of different subclasses uniformly. A list of Shapes can be iterated and area() called on each without knowing the specific type.

# Heron's formula for triangle area:
# s = (a + b + c) / 2
# area = math.sqrt(s * (s-a) * (s-b) * (s-c))

What to do

  • Create Triangle(Shape) with sides a, b, c, implementing area() using Heron's formula.
  • Create a list of shapes: [Circle(3), Rectangle(4, 5), Triangle(3, 4, 5)].
  • Print describe() for each.
  • Print the total area of all shapes, rounded to 2 decimal places: 'Total area: [value]'.

Watch out for

  • round(total, 2) may show 54.27 — floating-point arithmetic; use round() rather than :.2f for the total to get an exact value for the checker.
  • Do not hard-code the total — compute it by summing the areas.
Hints (4)
  • s = (a + b + c) / 2; return math.sqrt(s * (s-a) * (s-b) * (s-c))
  • Total: sum(s.area() for s in shapes)
  • f"Total area: {round(total, 2)}" formats the final line.
  • A 3-4-5 triangle has area 6.0 — you can verify this manually.
