Instance vs Class Variables

In this lesson, you will learn the difference between two types of variables inside a class. Some data belongs to each object. Some data is shared by all objects.

1. Instance Variables — Personal Data

An instance variable belongs to one specific object. Each object has its own copy.

Analogy
Think about students in a school. Each student has their own name and age. That information is personal. 
class Student:
    def __init__(self, name):
        self.name = name

student1 = Student("Arjun")
student2 = Student("Maya")

print(student1.name)
print(student2.name)

Explanation:

  • self.name is an instance variable.
  • student1 and student2 store different names.
  • Each object keeps its own data.

2. Class Variables — Shared Data

A class variable is shared by all objects of that class. There is only one copy.

Analogy
All students go to the same school. The school name is the same for everyone. That is shared information. 
class Student:
    school_name = "Greenwood High"

    def __init__(self, name):
        self.name = name

student1 = Student("Arjun")
student2 = Student("Maya")

print(student1.school_name)
print(student2.school_name)

Explanation:

  • school_name is written directly inside the class.
  • It is not inside __init__.
  • All objects share the same value.

3. Changing Class Variables

If we change a class variable, it affects all objects. 

Student.school_name = "Sunrise School"

print(student1.school_name)
print(student2.school_name)

Both objects now show the new school name.

But if we change an instance variable: 

student1.name = "Rahul"

print(student1.name)
print(student2.name)

Only student1 changes. student2 stays the same.

4. When to Use Each?

Use instance variables when the data is different for each object.

Use class variables when the data should be shared by all objects.

Simple Rule
If changing the value should affect everyone → use a class variable. If changing the value should affect only one object → use an instance variable.

5. Small Exercise

Create a class called Car.

  • Add a class variable called wheels with value 4.
  • Add an instance variable called brand.
  • Create two car objects with different brands.
  • Print both brand and wheels.

Try changing the class variable and see what happens.

6. What You Learned

You now understand:

  • Instance variables belong to one object.
  • Class variables are shared by all objects.
  • Where you define the variable matters.

Friendly Tip
Beginners often confuse these two. If your objects start sharing data by mistake, check whether you used a class variable instead of an instance variable.
