String Manipulation Challenges

Practise f-strings, slicing, and common string methods.

Part 1 — Slicing

Strings are sequences. s[start:stop:step] extracts a slice. Negative indices count from the end.

s = "Hello, World!"
print(s[0:5])    # Hello
print(s[-6:])    # World!
print(s[::-1])   # !dlroW ,olleH
Output:
Hello
World!
!dlroW ,olleH

What to do

  • Using sentence = "the quick brown fox", print just the word "quick" using a slice.
  • Print sentence reversed using a slice.
  • Print every other character of sentence starting from index 0.

Watch out for

  • Slicing is zero-indexed — count characters carefully.
  • The step -1 in [::-1] works on the full string — you do not need to specify start or stop.
Hints (5)
  • "quick" starts at index 4 and ends before index 9.
  • Reverse with [::-1].
  • Every other character: [::2]
  • To find the right indices, count characters from zero: t=0, h=1, e=2, (space)=3, q=4 ...
  • s[4:9] extracts characters at positions 4, 5, 6, 7, 8 — stop index is exclusive.

Part 2 — Methods & f-strings

String methods return new strings — they do not modify the original. Chain them freely.

s = "  hello world  "
print(s.strip().title())   # Hello World
print(s.strip().replace("world", "Python"))  # hello Python
Output:
Hello World
hello Python

What to do

  • Take sentence = "the quick brown fox".
  • Print it in UPPER CASE.
  • Print it in Title Case.
  • Replace "fox" with "cat" and print.
  • Print the number of times the letter 'o' appears.

Watch out for

  • Methods are case-sensitive — .Count() does not exist.
  • .title() capitalises the first letter of every word, including small prepositions — that's expected here.
Hints (4)
  • .upper(), .title(), .replace(), .count() are all you need.
  • .count('o') counts non-overlapping occurrences.
  • Methods do not modify sentence in place — sentence remains unchanged after each call.
  • You can call methods on the string literal directly or on the variable — both work.

Stretch — Caesar Cipher stretch

A Caesar cipher shifts each letter by a fixed number of positions in the alphabet, wrapping around from z back to a. Non-letter characters are left unchanged.

# ord('a') = 97, chr(97) = 'a'
# Shift 'a' by 3: ord('a') - 97 = 0; (0 + 3) % 26 = 3; chr(3 + 97) = 'd'

What to do

  • Write caesar(text, shift) that encodes text using a Caesar cipher.
  • Preserve case: uppercase letters shift within uppercase, lowercase within lowercase.
  • Leave spaces, digits, and punctuation unchanged.
  • caesar('Hello, World!', 3) → 'Khoor, Zruog!'
  • Print the result of caesar('Hello, World!', 3).

Watch out for

  • % 26 handles the wrap-around — 'x' shifted by 3 should give 'a', not a character outside the alphabet.
  • Do not shift non-letter characters — append them unchanged.
Hints (6)
  • Loop over each character with a for loop.
  • Use c.isalpha() to check if a character is a letter.
  • Use c.isupper() to branch between uppercase and lowercase shifting.
  • For uppercase: chr((ord(c) - ord('A') + shift) % 26 + ord('A'))
  • For lowercase: chr((ord(c) - ord('a') + shift) % 26 + ord('a'))
  • Build the result with a list and ''.join() for efficiency.
