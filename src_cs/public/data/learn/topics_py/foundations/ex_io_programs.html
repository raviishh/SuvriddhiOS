I/O Mini-Programs (Calculator, Greeter)

Use input() and print() together to build small interactive programs.

Part 1 — Greeter

input() reads a line of text the user types and returns it as a string. You can store it in a variable and use it immediately. 

name = input("What is your name? ")
print(f"Hello, {name}!")
Output:
What is your name? Alice
Hello, Alice!

What to do

  • Ask the user for their name using input().
  • Ask the user for their city using input().
  • Print: Hello, [name]! You live in [city].

Watch out for

  • The test will supply sample inputs — don't hard-code values.
  • Match the punctuation exactly: comma after the name, exclamation mark, then 'You live in'.
Hints (4)
  • Store each input() call in its own variable.
  • Build the output with an f-string.
  • The prompt string you pass to input() is optional but good practice — it tells the user what to type.
  • input() always returns a string, even if the user types a number.

Part 2 — Simple Calculator

input() always returns a string. To do maths you must convert it: int() for whole numbers, float() for decimals. 

a = int(input("Enter a number: "))
b = int(input("Enter another: "))
print(a + b)
Output:
Enter a number: 5
Enter another: 3
8

What to do

  • Ask the user for two numbers (use int() to convert each).
  • Print their sum on one line: Sum: [result]
  • Print their product on the next line: Product: [result]

Watch out for

  • Labels must match exactly: 'Sum: ' and 'Product: ' with a capital letter and a space after the colon.
  • Using float() instead of int() will produce 8.0 and 15.0 — the checker expects plain integers.
Hints (4)
  • Wrap input() inside int(): a = int(input("First number: "))
  • Use * for multiplication.
  • Both conversions must happen before you do any arithmetic — you can't add strings.
  • If you get a ValueError, check that int() is wrapping the entire input() call.

Stretch — Temperature Converter stretch

Real-world programs often need to branch based on user input. Combine input(), float conversion, and conditionals to handle multiple conversion directions. 

# Celsius to Fahrenheit: F = C * 9/5 + 32
# Fahrenheit to Celsius: C = (F - 32) * 5/9

What to do

  • Ask the user to enter a temperature value (convert to float).
  • Ask the user to enter the unit: C or F.
  • If C, convert to Fahrenheit and print: [value]C = [result]F
  • If F, convert to Celsius and print: [value]F = [result]C
  • Round both the input and result to 1 decimal place in the output.
  • Test with 100C (should give 100.0C = 212.0F) and 32F (should give 32.0F = 0.0C).

Watch out for

  • Don't forget .strip() on the unit input — a trailing space will break the comparison.
  • The formula uses integer division with Python 3 — 9/5 gives 1.8, not 1 (which it would in Python 2).
Hints (4)
  • Use unit = input(...).strip().upper() to handle lowercase input gracefully.
  • round(value, 1) rounds to one decimal place.
  • Use an if/elif block to branch on the unit.
  • f"{round(val, 1)}C = {round(result, 1)}F" formats the output cleanly.
