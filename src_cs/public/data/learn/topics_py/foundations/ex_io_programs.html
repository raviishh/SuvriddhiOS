Exercise 2  ·  Foundations

Input & Output

Programs that can listen and respond are far more useful. In this exercise you will build a short interactive Python program that takes information from the user, processes it, and prints a clear result.

Concepts practised
input() print() type conversion f-strings variables
Objective

Write a program that asks the user for their name and age, then prints a personalised message that includes their name and tells them how old they will be next year.

Tasks
  • 1 Use input() to ask the user: What is your name? Store the result in a variable called name.
  • 2 Use input() to ask: How old are you? Store the result in a variable called age. Convert it to an integer using int().
  • 3 Calculate next year's age by adding 1 to age. Store it in a variable called next_age.
  • 4 Use an f-string to print: Hello, [name]! You are [age] years old. Next year you will be [next_age].
Quick reference
Getting input
x = input("Ask something: ")

Always returns a string, even if the user types a number.
Converting types
int(x) → whole number
float(x) → decimal
str(x) → text
f-strings
f"Hello, {name}!"

Put an f before the quote, then use {variable} inside.
Printing
print("text")
print(variable)
print(f"Hi {name}")

Blue text shows what the user types. Your output will match whatever the user enters.

⚠️
Watch out — input() always gives you a string
If you try to do age + 1 without converting first, Python will raise a TypeError. You cannot add a number to a string. Always convert with int() or float() before doing maths.
💡
Combining input and conversion in one step
Instead of two lines, you can write age = int(input("How old are you? ")) — the inner call runs first and its result is immediately converted.
