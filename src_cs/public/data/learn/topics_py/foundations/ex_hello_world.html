Hello World Variants & Variable Juggling

Get comfortable with print() and variables by producing specific outputs.

Part 1 — Hello, World!

The print() function outputs text to the console. Pass it a string — text wrapped in quotes — and Python displays it.

print("Hello, World!")
Output:
Hello, World!

What to do

  • Print exactly: Hello, World!
  • Print exactly: I am learning Python.

Watch out for

  • The checker is case-sensitive and whitespace-sensitive.
  • Do not add extra blank lines between the two print() calls — they won't affect output, but get into the habit of clean code.
Hints (4)
  • Call print() once for each line — each call starts a new line automatically.
  • Watch capitalisation and punctuation — commas, exclamation marks, and periods all count.
  • Strings can use single quotes or double quotes interchangeably: print('Hello') and print("Hello") both work.
  • If you see a SyntaxError, check that every opening quote has a matching closing quote.

Part 2 — Variable Juggling

Variables store values. You can reassign a variable to a new value at any point, and Python will use the latest value.

x = 10
x = 20
print(x)
Output:
20

What to do

  • Create a variable called language and set it to "Python".
  • Create a variable called version and set it to 3.
  • Print: I love Python 3
  • Then reassign version to 4.
  • Print: I love Python 4

Watch out for

  • No extra spaces or punctuation — match the expected output exactly.
  • version is an integer (3), not a string ("3") — f-strings handle the conversion for you automatically.
Hints (4)
  • Use an f-string: f"I love {language} {version}"
  • Reassign version = 4 before the second print.
  • f-strings evaluate the variable at the moment print() is called — so reassigning version before the second print will automatically give you the new value.
  • You only need to define language once; only version changes.

Stretch — Multi-Variable Swap stretch

Python allows you to swap two variables in a single line using tuple unpacking: a, b = b, a. No temporary variable needed.

a = 'hello'
b = 'world'
a, b = b, a
print(a, b)  # world hello
Output:
world hello

What to do

  • Create three variables: first = 'alpha', second = 'beta', third = 'gamma'.
  • Using only one line of code (tuple unpacking), rotate them so first gets second's value, second gets third's value, and third gets first's value.
  • Print all three on one line separated by spaces.
  • Expected output: beta gamma alpha

Watch out for

  • Do not use a temporary variable — the point is to do it in one assignment line.
  • Order matters: first, second, third = second, third, first is correct; reversing the order gives a different rotation.
Hints (3)
  • Tuple unpacking: first, second, third = second, third, first
  • The right-hand side is evaluated completely before any assignment happens — Python captures the original values first.
  • This is a three-way rotation, not a two-way swap.
