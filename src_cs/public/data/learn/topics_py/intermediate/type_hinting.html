Type Hinting (typing)

In this lesson, you will learn how to add type information to your Python code. This is called type hinting.

1. What is type hinting?

Python does not require types. You can write: 

def add(a, b):
    return a + b

This works. But what should a and b be? Numbers? Strings? Lists?

Simple Idea
Type hints are labels. They tell humans (and tools) what kind of data is expected.

2. Adding basic type hints

We add type hints using a colon :. 

def add(a: int, b: int) -> int:
    return a + b

Explanation:

  • a: int means a should be an integer.
  • b: int means b should be an integer.
  • -> int means the function returns an integer.

Important: Python does not stop you if you pass the wrong type. Type hints are not enforced automatically.

3. Type hints for variables

You can also add type hints to variables. 

age: int = 18
name: str = "Alice"
price: float = 10.5
is_active: bool = True

These hints make your code clearer.

4. Using typing for lists and dictionaries

For more complex types, we use the typing module. 

from typing import List, Dict

numbers: List[int] = [1, 2, 3]
scores: Dict[str, int] = {"Alice": 90, "Bob": 85}

  • List[int] means a list of integers.
  • Dict[str, int] means keys are strings, values are integers.

5. Optional and Union

Sometimes a value can be more than one type. 

from typing import Optional, Union

def greet(name: Optional[str]) -> str:
    if name is None:
        return "Hello, Guest"
    return "Hello, " + name

def square(value: Union[int, float]) -> float:
    return value * value

  • Optional[str] means it can be str or None.
  • Union[int, float] means it can be int or float.

6. Why type hinting matters

Type hints:

  • Make code easier to read.
  • Help detect mistakes early.
  • Improve editor suggestions.

Important Idea
Type hints help humans first. They help tools second.

7. Try it yourself

Add type hints to this function: 

def multiply(a, b):
    return a * b
Challenge
Create a function that takes a list of integers and returns their total. Add correct type hints.

8. What you learned

Now you understand:

  • How to add type hints to functions.
  • How to hint variables.
  • How to use List, Dict, Optional, and Union.
  • Type hints are not enforced automatically.
