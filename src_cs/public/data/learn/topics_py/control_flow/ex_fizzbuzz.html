FizzBuzz Variations

Classic conditional + loop logic — with a twist.

Classic FizzBuzz (1–20)

For each number 1–20: print Fizz if divisible by 3, Buzz if divisible by 5, FizzBuzz if both, otherwise print the number.

for n in range(1, 6):
    if n % 15 == 0:
        print("FizzBuzz")
    elif n % 3 == 0:
        print("Fizz")
    elif n % 5 == 0:
        print("Buzz")
    else:
        print(n)
Output:
1
2
Fizz
4
Buzz

What to do

  • Print FizzBuzz for numbers 1 through 20 following the rules above.

Watch out for

  • Order matters — if you check % 3 before % 15 you'll print Fizz instead of FizzBuzz for 15.
  • Each number/word must be on its own line — do not use end=' ' here.
Hints (4)
  • Check divisibility by 15 (or both 3 and 5) FIRST, before checking 3 or 5 individually.
  • Use the % (modulo) operator: n % 3 == 0 means n is divisible by 3.
  • An alternative to checking n % 15 is: if n % 3 == 0 and n % 5 == 0 — both are equivalent.
  • range(1, 21) gives 1 through 20 inclusive — range(1, 20) would stop at 19.

FizzBuzz Sum

You can accumulate results while looping by maintaining a running variable outside the loop.

total = 0
for n in range(1, 6):
    total += n
print(total)  # 15
Output:
15

What to do

  • Sum all numbers from 1 to 100 that are NOT divisible by 3 or 5.
  • Print just the final sum.

Watch out for

  • Print only the number — no labels.
  • range(1, 100) stops at 99 — use range(1, 101) to include 100.
Hints (4)
  • Only add n to the total when n % 3 != 0 and n % 5 != 0.
  • Use range(1, 101) to include 100.
  • Alternatively: total = sum(n for n in range(1, 101) if n % 3 != 0 and n % 5 != 0)
  • Initialise total = 0 before the loop.

Stretch — Generalised FizzBuzz stretch

A generalised FizzBuzz accepts any number of (divisor, word) pairs and builds the output string dynamically. This eliminates hard-coded conditionals entirely.

# pairs = [(3, 'Fizz'), (5, 'Buzz'), (7, 'Bazz')]
# 21 → FizzBazz (divisible by 3 and 7)
# 35 → BuzzBazz (divisible by 5 and 7)
# 105 → FizzBuzzBazz (divisible by 3, 5, and 7)

What to do

  • Write fizzbuzz(n, pairs) where pairs is a list of (divisor, word) tuples.
  • For each number 1 to n, build the output string by concatenating the word for each matching divisor.
  • If no divisor matches, use the number itself.
  • Call fizzbuzz(20, [(3, 'Fizz'), (5, 'Buzz')]) and print each result — it must match classic FizzBuzz output.
  • Then call fizzbuzz(15, [(3, 'Fizz'), (5, 'Buzz'), (7, 'Bazz')]) and print — verify 14 prints FizzBazz.

Watch out for

  • The pairs must be checked in order — the order they appear in the list controls the output string.
  • Do not use a hard-coded 15 or any product — derive matches purely from the pairs list.
Hints (3)
  • Build result = '' then loop over pairs: if n % div == 0: result += word
  • After the loop: print(result if result else n)
  • This pattern scales to any number of rules without adding more if/elif branches.
