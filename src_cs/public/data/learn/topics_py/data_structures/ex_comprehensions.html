List & Dict Comprehension Drills

Rewrite loops as comprehensions and gain fluency with filtering.

Part 1 — List Comprehensions

[expr for item in iterable if condition] creates a new list. The if part is optional.

squares = [x**2 for x in range(1, 6)]
print(squares)  # [1, 4, 9, 16, 25]
Output:
[1, 4, 9, 16, 25]

What to do

  • Using numbers (1–20), create and print a list of all even numbers.
  • Create and print a list of squares of odd numbers only.
  • Create and print a list of strings: ['1 is odd', '2 is even', ...] for 1–5.

Watch out for

  • The third list uses range(1, 6), not all of numbers — read the instructions carefully.
  • Ternary syntax: value_if_true if condition else value_if_false — the condition sits in the middle.
Hints (4)
  • Even: [n for n in numbers if n % 2 == 0]
  • Odd squares: [n**2 for n in numbers if n % 2 != 0]
  • Use a ternary in the expression: f"{n} is {'even' if n % 2 == 0 else 'odd'}"
  • For the string list use range(1, 6) directly — numbers goes up to 20 but you only need 1–5 here.

Part 2 — Dict Comprehensions

{key: value for item in iterable} builds a dictionary in one expression.

words = ["cat", "dog", "elephant"]
lengths = {w: len(w) for w in words}
print(lengths)  # {'cat': 3, 'dog': 3, 'elephant': 8}
Output:
{'cat': 3, 'dog': 3, 'elephant': 8}

What to do

  • Create a dict mapping each number 1–5 to its cube.
  • Print the dict.

Watch out for

  • Dicts maintain insertion order in Python 3.7+ — print output should match the expected order.
  • range(1, 6) gives 1, 2, 3, 4, 5 — range(1, 5) would stop at 4.
Hints (3)
  • {n: n**3 for n in range(1, 6)}
  • n**3 is n cubed: 2**3 = 8, 3**3 = 27.
  • Dict comprehensions are identical in structure to list comprehensions but use {} and have a key: value expression.

Stretch — Nested Comprehension Matrix stretch

Comprehensions can be nested. A list comprehension whose expression is itself a list comprehension produces a 2D structure.

# 3×3 identity matrix via nested comprehension:
# [[1 if i == j else 0 for j in range(3)] for i in range(3)]
# → [[1,0,0],[0,1,0],[0,0,1]]

What to do

  • Using a single nested list comprehension (no for loops), create a 5×5 multiplication table as a list of lists.
  • table[i][j] should equal (i+1) * (j+1) for i, j in 0..4.
  • Print each row on its own line.
  • First row: [1, 2, 3, 4, 5]; last row: [5, 10, 15, 20, 25].

Watch out for

  • The outer comprehension produces the rows; the inner produces the columns — keep the order consistent.
  • table[0][0] should be 1 (1×1), table[4][4] should be 25 (5×5).
Hints (4)
  • Outer comprehension: [... for i in range(5)]
  • Inner comprehension: [(i+1)*(j+1) for j in range(5)]
  • Combined: [[(i+1)*(j+1) for j in range(5)] for i in range(5)]
  • To print each row: for row in table: print(row)
